# Public Folder - Logo Instructions

## Kako dodati logo:

1. **Stavi svoju logo sliku u ovaj folder** (`public/`)
2. **Preimenuj je u `logo.png`** (ili promijeni putanju u kodu ako koristiš drugi naziv)
3. **Podržani formati**: PNG, JPG, SVG, WebP

## Trenutna putanja:
- Logo se učitava sa: `/logo.png`
- To znači da slika mora biti u `public/logo.png`

## Alternativno:
Ako želiš koristiti drugi naziv ili format, promijeni u:
- `components/Hero.tsx` (linija ~68)
- `App.tsx` (linija ~120 i ~138)

## Preporučene dimenzije:
- Hero sekcija: 450px širina (veća slika)
- Navbar: 48px x 48px (manja slika)
- Format: PNG s transparentnom pozadinom (preporučeno)
